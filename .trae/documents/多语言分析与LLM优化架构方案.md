## 二、核心 Agent 设计

### 2.1 隐形知识挖掘 Agent

**职责**: 从代码中自动提取隐含的架构决策、设计模式、业务规则

**核心能力**:
1. **设计模式识别**
   - 识别单例、工厂、观察者、策略等模式
   - 检测 MVC、MVVM、分层架构等架构风格
   - 发现微服务边界和领域上下文

2. **业务规则提取**
   - 从注释中提取业务逻辑说明
   - 从命名约定推断业务概念
   - 从条件逻辑中提取业务规则

3. **隐性依赖发现**
   - 跨文件隐式调用关系
   - 配置驱动的依赖关系
   - 事件驱动架构中的发布-订阅关系

**文件规划**:
- `src/pywiki/agents/implicit_knowledge_agent.py` - 主 Agent 实现
- `src/pywiki/agents/pattern_recognizer.py` - 模式识别器
- `src/pywiki/agents/business_rule_extractor.py` - 业务规则提取器

### 2.2 记忆系统 Agent

**职责**: 智能管理知识，主动推荐相关记忆

**核心能力**:
1. **语义记忆索引**
   - 自动向量化所有记忆条目
   - 构建记忆关联图谱
   - 支持语义相似度搜索

2. **上下文感知检索**
   - 基于当前任务自动检索相关记忆
   - 记忆优先级动态调整
   - 跨项目记忆迁移和复用

3. **主动记忆推荐**
   - 在编码时主动推荐相关设计决策
   - 在架构设计时推荐历史方案
   - 学习用户偏好并优化推荐

**文件规划**:
- `src/pywiki/agents/memory_agent.py` - 主 Agent 实现
- `src/pywiki/agents/semantic_indexer.py` - 语义索引器
- `src/pywiki/agents/memory_recommender.py` - 记忆推荐器

### 2.3 架构洞见 Agent

**职责**: 深度分析架构，提供优化建议

**核心能力**:
1. **架构健康度评估**
   - 模块化程度分析
   - 耦合度和内聚性评估
   - 技术债务识别

2. **架构可视化**
   - 生成 C4 模型图
   - 领域驱动设计上下文映射
   - 数据流和依赖关系图

3. **架构建议生成**
   - 基于最佳实践的重构建议
   - 技术选型推荐
   - 性能优化建议

**文件规划**:
- `src/pywiki/agents/architecture_agent.py` - 主 Agent 实现
- `src/pywiki/agents/architecture_analyzer.py` - 架构分析器
- `src/pywiki/agents/architecture_advisor.py` - 架构顾问

### 2.4 多语言分析 Agent

**职责**: 统一分析多语言项目，提供跨语言视图

**核心能力**:
1. **统一代码模型**
   - 创建跨语言的统一 AST 表示
   - 支持语言间的调用链追踪
   - 识别跨语言的 API 契约

2. **语言特定增强**
   - Python: 类型推断、装饰器分析
   - TypeScript: 泛型分析、类型守卫识别
   - Java: 泛型擦除处理、注解元数据分析

**文件规划**:
- `src/pywiki/agents/multilang_agent.py` - 主 Agent 实现
- `src/pywiki/agents/unified_code_model.py` - 统一代码模型
- `src/pywiki/agents/cross_lang_analyzer.py` - 跨语言分析器

## 三、Agent 协调器设计

**文件**: `src/pywiki/agents/orchestrator.py`

**职责**: 协调多个 Agent 协同工作

**核心功能**:
1. **任务分发** - 根据任务类型分发给合适的 Agent
2. **结果聚合** - 整合多个 Agent 的分析结果
3. **工作流编排** - 定义 Agent 间的协作流程
4. **冲突解决** - 处理 Agent 间的冲突和重复
